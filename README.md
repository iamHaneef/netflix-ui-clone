# 🎬 Netflix Clone

A fully responsive Netflix UI clone built using HTML, CSS, and JavaScript.
This project recreates the front-end layout of Netflix, including the hero section, movie cards, hover effects, navigation bar, and smooth scrolling interactions.

📁 netflix-clone

 # ✨ Features
# ✅ 1. Responsive UI

Fully mobile, tablet & desktop friendly
Exact layout replication of Netflix homepage

# ✅ 2. Navigation Bar

Logo, menu items, profile icon
Sticky navbar on scroll

# ✅ 3. Hero Section

Big movie banner with title & description
Watch button + More Info button

# ✅ 4. Movie Rows (Carousels)

Scrollable horizontal movie lists
Smooth scroll using JavaScrip
Hover animations

# ✅ 5. Clean & scalable code structure

Well-organized CSS
Minimal & efficient JS
Simple to upgrade or add new movie sections
