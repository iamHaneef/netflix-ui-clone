# Netflix UI Clone

## 📌 Overview

A front-end clone of the Netflix homepage built using HTML, CSS, and JavaScript.  
This project focuses on UI design, layout structuring, and responsive styling.

---

## ⚡ Features

- Netflix-inspired UI layout  
- Responsive design  
- Styled components using CSS  
- Interactive elements using JavaScript  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
